# How to get these sites

```sh
sudo yum install git -y
git clone https://github.com/jijinmichael/AWS-ELB-Site.git
cd AWS-ELB-Site
```
Move any site to Website Docroot
> Eg:
```sh
cp -r ./* /var/www/html/         #<------ it depends on your webserver document root
```
